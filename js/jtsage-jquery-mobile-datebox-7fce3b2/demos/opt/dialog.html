jQueryMobile - DateBox

Home

A Date and Time Picker plugin for jQueryMobile

Forced Popup Mode

You can force the dialog mode off with 'useDialogForceFalse' set to 'true' (this is the default!!)

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useDialogForceFalse": true}'>

Forced Dialog Mode

You can force the dialog mode on with 'useDialogForceTrue' set to 'true' and 'useDialogForceFalse' set to 'false'

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useDialogForceTrue": true, "useDialogForceFalse": false}'>

Fullscreen (non-forced) Mode

On small displays (< 400px wide), this will use the full screen to display the picker, *without* using a dialog - 'fullScreen' set to 'true'

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"fullScreen": true}'>

Fullscreen (forced) Mode

This will use the full screen to display the picker, *without* using a dialog - 'fullScreen' and 'fullScreenAlways' set to 'true'

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"fullScreen": true, "fullScreenAlways": true}'>

Centered Mode

You can force DateBox to display centered on the page with 'centerWindow' set to 'true'.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"centerWindow": true}'>

Inline Mode

You can force DateBox to display inline with 'useInline' set to 'true'.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useInline": true}'>

Inline Blind Mode

You can force DateBox to display inline (with hiding) with 'useInline', 'useInlineBlind', and 'noButtonFocusMode' set to 'true'.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useInline": true, "noButtonFocusMode": true, "useInlineBlind": true}'>

Inline w/o Input

You can force DateBox to display inline (without the input hiding) with 'useInline' and 'useInlineHideInput' set to 'true'.

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"useInline": true, "useInlineHideInput": true}'>
