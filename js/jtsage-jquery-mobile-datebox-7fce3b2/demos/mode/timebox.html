A Date and Time Picker plugin for jQueryMobile

Time Default

Using the time mode is as simple as setting the 'mode' option to "timebox".

<label for="mydate">Some Time</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "timebox"}'>

12 Hour Clock Mode

To use a 12 hour clock instead or a 24 hour clock, set 'timeFomat' to 12.

<label for="mydate">Some Time</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "timebox", "timeFormatOverride": 12}'>
